修改陣列或物件的注意事項

常用語法：

  1. push: 新增數字
  2. pop: 減少數字
  3. shift: 移除並回傳陣列的第一個元素
  4. unshift: 從陣列開頭塞數字
  5. splice: 從頭數到第 X 個，移除 Y 個，加入 Z
  6. sort: 把亂掉的順序，按順序排列
  7. reverse: 順序顛倒 code
